# isumi

One-page portfolio in React, Vite, and Tailwind CSS, with a brutalist/cyber-sigilist aesthetic, a restricted ZINC palette, Space Mono, and the 泉 character as the platform icon.

## Scripts

- `npm run dev`: local server
- `npm run build`: production build
- `npm run preview`: build preview

## GitHub Pages

The workflow in `.github/workflows/deploy.yml` publishes the `dist` folder to GitHub Pages on pushes to `main`.

For a custom domain, configure it in GitHub Pages settings and add a `public/CNAME` file with the exact domain once it is defined.
